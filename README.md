**🌿 Crop Disease Detection**

Crop Disease Detection is a machine learning–based web application that helps identify plant diseases from crop leaf images. Users can upload a leaf image through the web interface, and the system analyzes it using a trained deep learning model to detect possible diseases.

The application classifies the leaf image and provides the predicted disease along with recommended solutions or treatments. This project demonstrates how computer vision and deep learning can be used in agriculture to assist farmers in early disease detection and better crop management.

**🚀 Features**
Upload crop leaf images for analysis
AI-based plant disease detection
Displays predicted disease name
Provides recommended treatments
Simple and user-friendly web interface


**🛠 Tech Stack**
Python
Django
TensorFlow / Keras
HTML, CSS, JavaScript
Deep Learning (InceptionV3 CNN Model)


**⚙️ Installation**
git clone https://github.com/hemanth-b-u/Crop-Disease-Detection.git
cd Crop-Disease-Detection
pip install -r requirements.txt
python manage.py runserver
Then open in browser:
http://127.0.0.1:8000/
